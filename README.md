# JARVIS Voice Assistant

## Setup
```bash
pip install -r requirements.txt
```

## Run
```bash
python jarvis.py
```

## Commands

**System Control:**
- Volume up/down/mute
- Brightness up/down
- WiFi on/off
- Screenshot
- Lock/Shutdown/Restart
- Show running apps

**App Control:**
- Open/close [app]

**Web & Knowledge:**
- Search [query]
- Play [video]
- What/Who/Where/When/How/Why [question] - Auto web search
- Tell me about [topic]

**Learning:**
- Auto-learns from statements ("My name is...")
- Correct me - Updates memory

**Conversation:**
- Always listening mode
- "Stop listening" - Pause (wake with "Hey Jarvis")
- "Stop/Exit" - Quit

## Note
Requires microphone access.
